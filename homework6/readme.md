这是一个家具城统计运营情况的可视化应用的Web前端，通过统计图表方式实时查看运营情况，引用了JQuery、Echart等第三方脚本库。
对这个Web应用主要做了3方面的优化工作，性能提高明显。
1、对应用的资源文件进行了压缩，主要是该应用的背景图"bg.png"原始文件有30M,在不影响视觉效果的情况下，改为"bg.jpg"文件实际大小为1.2M。在网络带宽不变的情况下，背景加载速度明显提升。
2、对主文件Index.html的内容进行了整理和优化，主要是将CSS和JavaScript整理成独立文件"Index.CSS"和"Index.js"，以引用形式加载。主文件和引用文件被先后读入，在带宽有限情况下，速度得到进一步优化。
3、进一步对"Index.js"代码内容进行了格式压缩处理，性能得到了进一步的优化。
